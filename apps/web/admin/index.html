Bảo mật Admin (khóa chuẩn)

  • Admin không dùng localStorage để giữ session. Chỉ HttpOnly cookie.
  • Không có “super admin” hardcode. Owner bootstrap qua secret.
  • Audit log bắt buộc: login, action, ip hash, timestamp.
  • Không cho phép upload tùy tiện: dùng R2 signed URL + virus scan (Stage 3+).
  • Nguyên tắc: “deny by default”.
Giai đoạn 1: UI. Giai đoạn 2: admin auth + D1 + audit.
