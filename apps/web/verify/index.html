Xác Minh

Xác minh theo lớp để giảm bot, giảm giả mạo, tăng tin cậy. Giai đoạn đầu là khung UI để khoá trải nghiệm, sau đó nối Worker API để cấp level.

Level 0
Khách. Xem công khai. Không nhắn tin hàng loạt.
Level 1
Email xác minh. Chống spam cơ bản. Rate-limit chặt.
Level 2+
Tăng quyền đăng bài, bán hàng, nhắn tin, gọi. Càng thật càng mở.
