# Security Policy — Muon Noi

## Reporting a Vulnerability
Email: security@muonnoi.org

Please include:
- What you found
- Steps to reproduce
- Proof-of-concept (minimal)
- Impact assessment

## Rules
- Do not exploit beyond what is necessary to demonstrate the issue.
- Do not access or download user data.
- Do not disrupt service.

## Scope
- app.muonnoi.org (Cloudflare Pages)
- future API Workers and related subdomains
